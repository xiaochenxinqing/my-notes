# Vue项目命名规范

**Vue项目命名规范**

文件：

- index.js 或者 index.vue，统一使用小写字母开头的**(****kebab-case****)**命名规范
- **属于****组件****或****类****的**，统一使用大写字母开头的**(****PascalCase****)**命名规范
- 其他非组件或类的（如页面文件），统一使用小写字母开头的**(****kebab-case****)**命名规范

文件夹：

    使用小写字母开头的**(****kebab-case****)**命名规范

组件名（**name**）：

    统一使用大写字母开头的**(****PascalCase****)**命名规范 或 使用小写字母开头的**(****kebab-case****)**    ，只能同时选择一种
