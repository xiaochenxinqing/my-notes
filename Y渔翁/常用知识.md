###  DER 和 PEM 的区别

| 特性              | DER                                | PEM                               |
| ----------------- | ---------------------------------- | --------------------------------- |
| **编码方式**      | 二进制编码                         | Base64 编码的文本                 |
| **可读性**        | 不可读（纯二进制格式）             | 可读（但内容是 Base64 编码）      |
| **使用场景**      | 常用于 Windows、移动设备和硬件     | 常用于 Unix/Linux 系统和网络传输  |
| **文件扩展名**    | `.der`, `.cer`                     | `.pem`, `.crt`, `.key`, `.cer`    |
| **文件头/尾标签** | 无                                 | 有明确的头部和尾部标签            |
| **传输方式**      | 二进制文件传输                     | 可以通过文本协议（如邮件）传输    |
| **大小**          | 通常比 PEM 更小（由于没有 Base64） | 较大（Base64 编码使数据变大 33%） |
