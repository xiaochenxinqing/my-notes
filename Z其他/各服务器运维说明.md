#########远程登录设置

1设定远程登录方式（禁用密码登录）

sudo  vim /etc/ssh/sshd_config  后 改为如下配置：

UseDNS yes

AddressFamily inet

PermitRootLogin no

SyslogFacility AUTHPRIV

PasswordAuthentication no   禁用密码登录

RSAAuthentication yes

AllowUsers hc git  root

PubkeyAuthentication yes   使用公钥登录

   

修改完毕后使用service sshd reload 使生效

2修改允许远程登录当前用户的客户端列表：

sudo  vim ~/.ssh/authorized_keys

#########git配置说明(此已弃用，改为gitblit)
